=====================================================
PostgreSQL Database Setup Commands for E-Commerce App
=====================================================

STEP 1: Create the Database
---------------------------
Run this command in CMD or PowerShell:

"C:\Program Files\PostgreSQL\18\bin\psql.exe" -U postgres -c "CREATE DATABASE ecommerce_db;"

If you get a password prompt, enter your PostgreSQL postgres user password.


STEP 2: Create Tables and Insert Data
--------------------------------------
Option A - Using SQL file (if you have create_postgres_db.sql):
"C:\Program Files\PostgreSQL\18\bin\psql.exe" -U postgres -d ecommerce_db -f "F:\React\omar aslan\e-commerce-s\database\create_postgres_db.sql"

Option B - Run SQL commands directly:
"C:\Program Files\PostgreSQL\18\bin\psql.exe" -U postgres -d ecommerce_db -c "DROP TABLE IF EXISTS products CASCADE; DROP TABLE IF EXISTS categories CASCADE;"

Then continue with the rest of the commands...


STEP 3: Verify Database Creation
---------------------------------
"C:\Program Files\PostgreSQL\18\bin\psql.exe" -U postgres -d ecommerce_db -c "SELECT COUNT(*) FROM categories; SELECT COUNT(*) FROM products;"
